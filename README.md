# Java Calculator

## Description
A console-based calculator developed in Java that performs basic arithmetic operations using switch-case statements.

## Features
- Addition
- Subtraction
- Multiplication
- Division
- Division by zero handling

## Technologies Used
- Java
- Scanner Class
- Switch Case

## How to Run

1. Compile:
   javac Calculator.java

2. Run:
   java Calculator
